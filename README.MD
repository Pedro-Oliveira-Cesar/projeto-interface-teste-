# Projeto Interface

## Telas Implementadas
- `index.html` (Página inicial)
- `listagem.html` (Lista de itens)
- `detalhe.html` (Detalhe de item)

## Decisões de Layout
- Estrutura semântica com `header`, `main`, `footer`.
- Layout mobile-first, utilizando Flexbox para ajustes em telas maiores.
- Escala tipográfica e espaçamento definidos em variáveis no `:root`.

## Breakpoints Utilizados
- `max-width: 420px` → Ajuste para celulares pequenos.
- `min-width: 768px` → Ajuste para tablets.
- `min-width: 1024px` → Ajuste para notebooks/desktops.

## Acessibilidade
- Contraste mínimo seguindo WCAG AA.
- Links possuem foco visível.
- Imagens possuem atributo `alt`.
